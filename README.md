## First steps
* Create configuration file in the root folder (config.json). E.G.:

```json
{}
```

* Install dependencies:

```bash
npm install
```

* Run the tests:

```bash
npm test
```

* Compile the code:

```bash
grunt
```

* Run the server:

```bash
npm start
```

### Watch for changes

```bash
grunt watch
```
